01分数规划

发表于 2019-10-13 更新于 2019-10-17 分类于 OI

01分数规划是二分答案中的一类常见问题

问题描述：

给你n对元素vi,wi,要求从中挑选出k对，使得，v的和除以w的和最大

解析：

假设我们已经求得了这个最大的值 x = sigma v [ i ] / sigma w [ i ] ;

则有

sigma(v [ i ] ) / sigma(w [ i ] ) >= x -> sigma(v [ i ] - x * w [ i ] ) >= 0

在数据确定的情况下，这个式子具有单调性，所以我们可以考虑二分求解这个x值

经典例题：

POJ 3111 K Best
