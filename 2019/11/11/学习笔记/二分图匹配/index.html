基础二分图

发表于 2019-11-11 分类于 OI

二分图是一种常见的图论模型

二分图基础知识

  • 二分图可以分成左右两个点集，所有的边都连着$U
    $到$V$

  • 二分图中没有奇环

  • 匹配：是一些边的集合，任意两条边没有公共顶点

  • 最大匹配：所有匹配中边数最多的匹配

  • 完美匹配：所有的点都在匹配中

  • 二分图最大匹配算法：匈牙利算法

  • 匹配点：盖点|非匹配点：未盖点

匈牙利算法

假设已经有了一个匹配，比较和最大匹配的差距

连接两端未盖的匹配边

然后在原图基础上找增广路，找不到说明是最大匹配

从一侧的未盖点走增广路，也就是匹配边|非匹配边|匹配边|非匹配边的走法，如果可以走到另一侧的未盖点，那么就找到了一条增广路

已经匹配的点永远不会退出匹配，只会更换匹配
